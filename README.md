<h1>👨‍💼 HR Analytics Dashboard (Power BI)</h1>
An interactive Power BI dashboard designed to analyze workforce data and provide insights into employee attrition, department performance, salary trends, workforce demographics, and job satisfaction. The dashboard enables HR professionals and business leaders to make data-driven decisions for improving employee retention and organizational performance.

<h2>📌 Project Overview / Purpose</h2>

The HR Analytics Dashboard is a comprehensive Power BI report built to monitor and analyze employee-related metrics across multiple dimensions such as attrition, salary, tenure, performance, promotions, demographics, and job satisfaction.

This dashboard helps organizations identify workforce trends, understand employee behavior, and support strategic HR planning.

<h2>🛠 Tech Stack</h2>
The dashboard was built using the following tools and technologies:
<br>
<br>
• 📊 Power BI Desktop – Main platform for dashboard creation <br>
• 📂 Power Query – Data cleaning and transformation <br>
• 🧠 DAX (Data Analysis Expressions) – KPIs and calculated measures <br>
• 🧩 Data Modeling – Relationships between sales, products, and transactions <br>

<h2>✨ Features & Dashboard Highlights
🔍 Business Problem </h2>

Organizations often struggle to understand:

•  Why employees leave the company? <br>
•  Which departments experience the highest attrition? <br>
•  How employee demographics affect workforce performance? <br>
•  Which job roles generate the highest payroll expenses? <br>
•  How satisfied employees are across departments? <br>

Without proper analytics, workforce planning becomes difficult and costly.

<h2>🎯 Goal of the Dashboard </h2>

To create an interactive HR analytics solution that:

•  Monitors employee attrition and retention <br>
•  Evaluates department-level performance <br>
•  AAnalyzes workforce demographics <br>
•  Tracks salary and promotion metrics <br>
•  Supports strategic HR decision-making <br>
